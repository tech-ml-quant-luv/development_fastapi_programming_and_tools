# What is object-oriented programming?
    Object-Oriented Programming(OOP) is a software programming paradigm based on organizing code around "objects", data fields and behaviours, rather that n just function and logic. It binds data and functions together into units to increase the code reusability, modularity and maintainability, often used for modelling real-world entities.
    Object-oriented programming tried to model a real world object.
    Every object has two things - ATTRIBUTES - what is has(A variable basically), METHODS - what it can do(A function basically)

## Package

A Python package is a way to organize related modules(single Python files) into a structured hierarchy or directories, creating a common names namespace/

### Class
A class is a blueprint or template used to create objects. It defines the common properties (attributes) and behaviours (methods) that all objects of a specific type will share.

    car = CarBlueprint() 
    In this case , car is an object that is made using the class/blueprint CarBlueprint.

### Attribute

An attribute is a variable that is associated with an object.
    
    self  - It is the actual opject that is being initialized. It contains all the data of the object including attributes and methods once an object is made with the class. Is is used to pass information and global state of the class to each method
